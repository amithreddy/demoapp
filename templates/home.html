Animal Image repository

How this was made

I, Amith Reddy, made this image repository to demonstrate my skillset and understanding of webservices.
This repo was made with python and a sqlite database. This site is hosted on a linux server which is configured to host a website.

Features

This is a image repository of animals. You can delete the pictures of the animals you don't like. Refer to the instructions below.

Instructions

  • To open a imageClick on the image name to open the image in a new tab.
  • To delete image(s) Select the checkbox and click the "Delete selected" button.
  • To reset the database to the original. Click the "Reset Database" button
